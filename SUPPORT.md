# Support

Use the channel that matches the problem:

- **Installation, configuration, and questions:** [GitHub Discussions Q&A](https://github.com/ajhcs/codex-edge-glow/discussions/categories/q-a)
- **Ideas and early concepts:** [GitHub Discussions Ideas](https://github.com/ajhcs/codex-edge-glow/discussions/categories/ideas)
- **Reproducible bugs:** [Bug report form](https://github.com/ajhcs/codex-edge-glow/issues/new?template=bug.yml)
- **Security or privacy vulnerabilities:** [Security policy](SECURITY.md)

Before posting, check [Installation](docs/installation.md), [Codex integration](docs/integrations.md), and [Troubleshooting](docs/troubleshooting.md).

When asking for help, include the app version, Windows build, display resolution/scaling, chosen pattern/island mode, reproduction steps, and whether live preview shows the same problem. Remove task text, usernames, folder paths, credentials, and unrelated desktop content from screenshots or recordings.
